# Stock Portfolio Manager

A full-stack web application that allows users to track stocks, manage portfolios, and monitor investments.

## Tech Stack

Frontend:
- React.js
- Axios
- React Router

Backend:
- Node.js
- Express.js
- JWT Authentication

Database:
- MongoDB

## Features

- User Registration & Login
- Secure Authentication with JWT
- Track stock portfolios
- Add / remove stocks
- View portfolio performance
- Admin dashboard
- REST API for stock data

## Installation

Clone the repository

```
git clone https://github.com/poojasukhdeve-project/stock-portfolio-manager.git
```

### Install backend

```
cd server
npm install
npm start
```

### Install frontend

```
cd client
npm install
npm start
```
